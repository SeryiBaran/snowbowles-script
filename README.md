## How to use

### Easy way

- Write simple ascii javascript code to file `./code_for_generateScript.js`
- Edit program name (what contains only 3 ascii symbols) in file `./src/generateScript.ts`
- Run `pnpm generateScript`
- Run `pnpm start`
- Run `node one.js`
